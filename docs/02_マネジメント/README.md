# マネジメントって何？

**チームの成果を最大化するための役割**らしい。
今までの弊チームは、リーダー（マネージャー）がチームメンバーをまとめて、設計を考えて、車を作って、仕事を振って...とほとんどの業務を担っていました。
このことを**作業の属人化**といいます。

# 仕事の頼み方

マネジメントとは仕事の頼み方。ちゃんと仕事を頼めるようになったら一流のマネージャーだ。

## どう仕事を頼むべきか？

**わからない。**外的要因に寄りすぎて、最適解がない....としか言いようがない
とはいえ、ある程度は具合がある。
