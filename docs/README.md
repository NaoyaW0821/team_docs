# このdocsについて

このdocsは、チームの決まったこと、考え方、判断理由、ノウハウを残すための場所です。

## 基本ルール
- Discordで議論 → 決まったらdocsに反映
- docsに書いてあることが正式決定
- 迷ったら ”00_思想と原則” に戻る


## どこに何を書くか
- 思想・価値観 → [00_思想と原則](./00_思想と原則/README.md)
- 技術的な判断理由 → [01_設計判断](./01_設計判断/README.md)
- 運営・進め方 → [02_マネジメント](./02_マネジメント/README.md)
- 日々の試行錯誤 → [03_実装ログ](./03_実装ログ/README.md)
- データ・辞書 → [04_リファレンス](./04_リファレンス/README.md)
